Main Features
1.	Face Recognition
-	ตรวจสอบตัวตนผู้ใช้งานด้วยการสแกนใบหน้า(face-api.js)
2.	Location Verification
-	อนุญาตให้ลงเวลาได้เฉพาะในรัศมี ไม่เกิน 200 เมตร
3.	Clock In / Clock Out
-	รองรับการลงเวลา
o	Morning
o	Lunch
o	Afternoon
o	Evening
4.	Attendance Records
-	เก็บประวัติการลงเวลาของผู้ใช้แต่ละคน
5.	Admin Panel
-	ผู้ดูแลสามารถตรวจสอบข้อมูลการลงเวลาได้
